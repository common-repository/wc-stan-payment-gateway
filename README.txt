Vos clients réalisent les paiements en passant par l'application Stan où toutes leurs informations sont pré-enregistrées et sécurisées, vous n'avez plus besoin de gérer la validité des informations du client lors du checkout.

= Comment fonctionne Stan : Fast pay ? =

Lors d'une commande, le client sélectionne le bouton "Payer avec Stan" comme mode de paiement. Le client ne remplit aucune information. Toutes ses informations identité, emails, livraisons sont automatiquement remplies avec la solution [Stan easy connect](https://wordpress.org/plugins/stan-easy-connect/).

Après avoir passé la commande, le client recevra un email de confirmation.

== Service ==

Stan Payment repose sur son service https://api.stan-app.fr. [Voir la documentation technique](https://doc.stan-app.fr).

Les paiements sont créés par nos prestataires agréés par l'ACPR.

Lors d'un paiement nous collections les informations suivantes :
+ Le montant du paiement
+ L'identifiant de l'ordre
+ L'identifiant du paiement (ID de transaction)

== Installation ==

# Installer la solution Stan en vidéo

[youtube https://www.youtube.com/watch?v=250SQjsT2JI]

# DEPUIS VOTRE COMPTE ADMIN WORDPRESS

Sur https://votre-site.fr/wp-admin

1. Allez sur Extensions > Ajouter
2. Cherchez Stan Payment Gateway, trouvez et installez Stan Payment Gateway.
3. Activez le plugins depuis la page Extensions

# DEPUIS WORDPRESS.ORG

1. Téléchargez Stan Payment Gateway
2. Dé-zippez and téléversez le dossier stan-payment-gateway dans `/wp-content/plugins/directory`
3. Activez Stan Payment Gateway depuis la page des plugins

== F.A.Q. ==

= Comment les utilisateurs paient avec Stan Payment Gateway ? =

Vos clients utilisent l'application Stan pour faire leur achat. Tous les paiements sont faits de banque à banque, par virement bancaire.

= Comment je reçois l'argent ? =

Lorsque vous configurez le plugin Stan Payment vous renseignez vos informations bancaires avec votre IBAN. Le montant lors d'un achat est directement viré vers votre compte bancaire.

= Est-ce sécurisé ? =

Les paiements réalisés avec Stan suivent les standards de la DSP2 grâce à notre partenaire agréé par l'ACPR.

= Qu'est-ce que Stan Easy Connect? =

[Stan Easy Connect](https://wordpress.org/plugins/stan-easy-connect/) est un plugin que nous proposons pour améliorer l'expérience de vos clients lors de l'inscription et lors du checkout.
